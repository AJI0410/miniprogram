# TabBar 图标文件

请在此目录下添加以下图标文件：

## 需要的图标文件：
- `home.png` - 首页图标（未选中状态）
- `home-active.png` - 首页图标（选中状态）
- `music.png` - 音乐图标（未选中状态）
- `music-active.png` - 音乐图标（选中状态）
- `workout.png` - 运动图标（未选中状态）
- `workout-active.png` - 运动图标（选中状态）

## 图标要求：
- 尺寸：建议 81px × 81px
- 格式：PNG
- 背景：透明
- 未选中状态：建议使用灰色 (#999999)
- 选中状态：建议使用绿色 (#07C160)

## 临时解决方案：
如果暂时没有图标，可以：
1. 使用微信开发者工具内置的图标
2. 从网上下载免费图标
3. 使用图标生成工具创建简单图标
